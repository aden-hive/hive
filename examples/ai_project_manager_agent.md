# AI Project Manager Agent

## Purpose
This agent converts meeting notes into structured project tasks.

## Workflow
Meeting Notes → Agent Analysis → Task Extraction → Structured Output

## Input Example

Team meeting notes:

Rachel will prepare the marketing strategy by April 10.
Uche will finalize product specifications by March 30.
Faruk will coordinate sales outreach materials.

## Output Example

Task: Prepare marketing strategy  
Owner: Rachel  
Deadline: April 10  

Task: Finalize product specifications  
Owner: Uche 
Deadline: March 30  

Task: Coordinate sales outreach materials  
Owner: Faruk 
Deadline: Not specified

## Use Case

Many project teams struggle to convert meeting discussions into clear tasks.  
This agent helps automate that step by transforming unstructured meeting notes into structured tasks that can be added to project management tools.
