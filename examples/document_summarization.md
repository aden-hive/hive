# Document Processing & Summarization Workflow

## Overview
This example demonstrates how Hive can be used to process and summarize documents using multiple agents working together.

## Workflow

1. **Input Agent**
   - Accepts document input (text/file)
   - Prepares data for processing

2. **Extraction Agent**
   - Extracts relevant text/content from the document

3. **Summarization Agent**
   - Generates a concise summary of the extracted content

4. **Formatting Agent**
   - Structures the output into a clean, readable format

## Output
A structured summary of the document, ready for use in reports or decision-making.

## Why This Matters
- Demonstrates real-world use of multi-agent orchestration
- Shows how Hive can automate document-heavy workflows
- Useful for enterprise use cases like reporting, research, and knowledge management
