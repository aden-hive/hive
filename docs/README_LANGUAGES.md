# Hive README Translations

Hive provides documentation in multiple languages.

Available README files:

- English: `README.md`
- Español (Spanish): `READMEes.md`
- 日本語 (Japanese): `READMEja.md`
- 한국어 (Korean): `READMEko.md`
- Português (Portuguese): `READMEpt.md`
- Русский (Russian): `READMEru.md`
- 简体中文 (Simplified Chinese): `READMEzh-CN.md`

If you would like to contribute a new translation, please follow the same naming convention and open a pull request.
