### Available Excel Tools

- `excel_read`: Read sheet data (with limit/offset)
- `excel_write`: Create new Excel file
- `excel_append`: Append rows to existing file
- `excel_info`: Get file metadata (sheets, columns, row counts)
- `excel_sheet_list`: List all sheet names
- `excel_sql`: Run SELECT queries via DuckDB
- `excel_search`: Search for values across sheets
- `excel_add_sheet`: Add new sheet
- `excel_rename_sheet`: Rename existing sheet
- `excel_delete_sheet`: Delete sheet
- `excel_update_cell`: Update single cell value
- `excel_delete_row`: Delete specific data row
- `excel_add_column`: Add new column with default value
- `excel_sort`: Sort sheet by one or more columns
