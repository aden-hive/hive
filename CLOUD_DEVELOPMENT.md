# Cloud Development Guide

## Why Cloud Development?
Aden focuses on the Coding Agent workflow where AI agents build other agents. Cloud development environments allow you to start using the Coding Agent immediately without local setup headaches like Python version conflicts, PEP 668 errors, or missing dependencies.

## GitHub Codespaces
### Quick Start
1. Navigate to the Hive repository
2. Click the green **"Code"** button
3. Select the **"Codespaces"** tab
4. Click **"Create codespace on main"**

### Pre-configured Environment
Your codespace comes with:
- Python 3.11+ pre-installed
- Node.js 18+ ready
- All required dependencies
- VS Code with extensions
