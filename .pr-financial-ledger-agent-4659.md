## Description

This PR adds a new **Financial Ledger Agent** that automates monthly transaction reporting from emails. The agent scans Gmail for financial transaction alerts, extracts and categorizes transaction data using LLM analysis, and generates structured monthly financial summaries in Excel and PDF formats.

### Key Features

- **Email Monitoring**: Scans Gmail inbox for financial triggers (Transaction Alert, Salary Credited, Invoice, Payment Received, etc.)
- **Information Extraction & Categorization**: Uses LLM to parse unstructured email content and automatically categorize transactions
- **Data Compilation**: Aggregates transactions and deduplicates overlapping alerts
- **Report Generation**: Creates Excel ledger and HTML summary reports

### Transaction Categories

The agent intelligently classifies transactions into categories:
- Salary, Food & Beverages, Fuel, Transportation, Internet, Utilities, Health, Travel, Entertainment, Shopping, Electronics, Subscription, Bill Payment, Transfer, Other

### Architecture

**6-Node Pipeline:**

| Node | Type | Description |
|------|------|-------------|
| `intake` | event_loop (client-facing) | Receives date range, currency preference, and max emails from user |
| `email-fetcher` | event_loop | Fetches financial emails from Gmail using search queries |
| `finance-classifier` | event_loop | Extracts and categorizes transactions using LLM |
| `data-aggregator` | event_loop | Aggregates and deduplicates transactions |
| `excel-generator` | event_loop | Creates Excel ledger file |
| `pdf-summarizer` | event_loop (client-facing) | Generates HTML summary and serves files to user |

**Edge Flow:**
```
intake → email-fetcher → finance-classifier → data-aggregator → excel-generator → pdf-summarizer
                                                                          ↓
                                                                    (loops back to intake for new scan)
```

**Tools Used:**
- `gmail_list_messages`, `gmail_batch_get_messages` - Gmail API
- `load_data`, `save_data`, `append_data`, `serve_file_to_user` - Data tools
- `excel_write` - Excel tool
