# Q and A Agent (Python)

This agent is designed to answer questions understandably and clearly.
It is a simple example of a Hive agent implemented in Python.

## Usage

You can run this agent using the Hive CLI:

```bash
./hive run exports/qanda_python --input '{"question": "What is the capital of France?"}'
```

## Goal

The goal of this agent is to provide clear and helpful answers to user questions, avoiding jargon where possible.
