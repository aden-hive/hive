# Resume Screening Agent

This example demonstrates a resume analysis agent built using the Hive framework.

The agent analyzes a candidate resume and extracts important information such as skills, experience, education, and provides a short evaluation summary.

## Goal

Analyze a resume and generate a structured summary of the candidate profile.

## What this example demonstrates

- How to adapt a Hive template into a resume analysis agent
- How agents can process text documents and extract structured information
- How to build HR automation agents using Hive

## Example Input

Resume text describing a candidate's skills, experience, and education.

## Expected Output

The agent should generate a structured response including:

1. Candidate Summary
2. Key Skills
3. Experience Summary
4. Education
5. Overall Evaluation

## Example Use Cases

- Resume screening
- Candidate evaluation
- HR automation
- Talent analysis
