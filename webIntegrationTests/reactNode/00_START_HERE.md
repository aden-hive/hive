## ⚡ 5-Minute Setup

```bash
# 1. Install backend
cd hive-web-dashboard/backend
npm install

# 2. Install frontend
cd ../frontend
npm install

# 3. Start backend (Terminal 1)
cd ../backend
npm run dev
# Runs on http://localhost:5000

# 4. Start frontend (Terminal 2)
cd ../frontend
npm run dev
# Runs on http://localhost:5173

# 5. Open dashboard
# http://localhost:5173
```

Done! 🎉
